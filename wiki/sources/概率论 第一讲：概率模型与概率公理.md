# 概率论 第一讲：概率模型与概率公理

原始资料：[[raw/articles/概率论_第一讲_概率模型与概率公理|概率论_第一讲_概率模型与概率公理]]

## 简短总结

这一讲是 [[概率论]] 的起点，重点不是立刻计算概率，而是先学会建立 [[概率模型]]：确定 [[样本空间]]，定义 [[事件]]，再用 [[概率律与概率公理]] 分配概率。

核心直觉：

> 概率论的第一步是建模：先确定所有可能结果，再定义事件，最后分配概率。

## 关键论点

- [[随机试验]] 是试验前无法确定具体结果、但知道所有可能结果的试验。
- [[样本空间]] $\Omega$ 是所有可能结果组成的集合。
- [[事件]] 是样本空间的子集，表示我们关心的一类结果。
- [[概率模型]] 通常写成 $(\Omega, P)$，其中 $\Omega$ 是样本空间，$P$ 是概率律。
- 概率律必须满足非负性、归一性和可加性三条公理。
- 补事件公式和一般并集公式是由概率公理推出的基础工具。
- 等可能模型只有在基本结果等可能时才可使用。

## 核心公式

| 名称 | 公式 | 含义 |
|---|---|---|
| 非负性 | $P(A)\ge 0$ | 概率不能为负 |
| 归一性 | $P(\Omega)=1$ | 样本空间必然发生 |
| 互斥可加性 | $P(A\cup B)=P(A)+P(B)$ | 互斥事件概率相加 |
| 补事件 | $P(A^c)=1-P(A)$ | 通过反面求概率 |
| 一般并集 | $P(A\cup B)=P(A)+P(B)-P(A\cap B)$ | 避免重复计算交集 |
| 等可能模型 | $P(A)=\frac{\#A}{\#\Omega}$ | 有利结果数 / 总结果数 |

## 易错点

- 样本空间不完整。
- 基本结果之间互相重叠。
- 随便假设基本结果等可能。
- 混淆具体结果和事件集合。

## 例题脉络

- 掷骰子：练习事件、交集、并集和一般并集公式。
- 连续抛硬币：用补事件求“至少一次”的概率。
- 合法概率模型判断：检查所有基本结果概率是否非负且总和为 1。
- 扑克牌：用并集公式处理“红桃或 A”。
- 抽球：在等可能模型下用有利结果数 / 总结果数计算概率。
