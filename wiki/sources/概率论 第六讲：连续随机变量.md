# 概率论 第六讲：连续随机变量

原始资料：[[raw/articles/概率论_第六讲_连续随机变量|概率论_第六讲_连续随机变量]]

## 简短总结

这一讲从离散世界进入连续世界。核心变化是：[[连续随机变量]] 的单点概率为 0，概率不再来自“点”，而来自 [[概率密度函数 PDF]] 曲线下的**面积**。描述工具从 [[概率质量函数 PMF]] 换成 PDF 和 [[累积分布函数 CDF]]。

从离散的：

$$
P(X=x)
$$

切换到连续的：

$$
P(a<X<b)=\int_a^b f_X(x)\,dx
$$

## 关键论点

- [[连续随机变量]] 可取区间上任意实数，单点概率 $P(X=x)=0$，但这不代表该点不可能取到。
- [[概率密度函数 PDF]] 不是概率，密度值可大于 1；概率是面积（积分）。
- 合法 PDF 必须非负且总面积为 1。
- [[累积分布函数 CDF]] 定义为 $F_X(x)=P(X\le x)$，是 PDF 的积分；PDF 是 CDF 的导数。
- 对连续随机变量，区间端点是否包含不影响概率。
- 三个核心连续分布：[[均匀分布]]、[[指数分布]]、[[正态分布]]。
- [[指数分布]] 具有 [[无记忆性]]，是离散 [[Geometric 分布]] 的连续对应物。
- [[正态分布]] 的 $N(\mu,\sigma^2)$ 第二个参数是方差；可用标准化 $Z=\frac{X-\mu}{\sigma}$ 转成标准正态。

## 核心公式

| 名称 | 公式 | 含义 |
|---|---|---|
| PDF 区间概率 | $P(a<X<b)=\int_a^b f_X(x)\,dx$ | 概率是面积 |
| PDF 总面积 | $\int_{-\infty}^{+\infty}f_X(x)\,dx=1$ | 总概率为 1 |
| CDF | $F_X(x)=P(X\le x)$ | 累积概率 |
| CDF 与 PDF | $F_X(x)=\int_{-\infty}^x f_X(t)\,dt$ | CDF 是面积 |
| PDF 与 CDF | $f_X(x)=F_X'(x)$ | PDF 是 CDF 导数 |
| Uniform PDF | $f_X(x)=\frac{1}{b-a}$ | $a\le x\le b$ |
| Exponential PDF | $f_X(x)=\lambda e^{-\lambda x}$ | $x\ge0$ |
| Exponential CDF | $F_X(x)=1-e^{-\lambda x}$ | $x\ge0$ |
| Exponential tail | $P(X>x)=e^{-\lambda x}$ | 等待超过 $x$ |
| Normal PDF | $\frac{1}{\sqrt{2\pi}\,\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$ | 钟形曲线 |
| 标准化 | $Z=\frac{X-\mu}{\sigma}$ | 转成标准正态 |

| 内容 | [[离散随机变量]] | [[连续随机变量]] |

| 描述函数 | [[概率质量函数 PMF]] | [[概率密度函数 PDF]] |
