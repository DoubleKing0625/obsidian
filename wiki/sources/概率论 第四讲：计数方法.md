# 概率论 第四讲：计数方法

原始资料：[[raw/articles/概率论_第四讲_计数方法|概率论_第四讲_计数方法]]

## 简短总结

这一讲处理 [[等可能模型]] 中最实际的问题：概率常写成“有利结果数 / 总结果数”，但这些结果数应该如何数。核心是把概率问题转化为 [[计数方法]] 问题。

$$
P(A)=\frac{\#A}{\#\Omega}
$$

## 关键论点

- [[加法原则]] 适合分类讨论，前提是各类情况不重叠；有重叠时要减去重复部分。
- [[乘法原则]] 适合分步骤完成，一个步骤的选择和另一个步骤组合起来。
- [[阶乘]] 是排列和组合公式的基础。
- [[排列]] 适合“有顺序、无放回”的选择。
- [[组合]] 适合“无顺序、无放回”的选择。
- [[可重复组合]] 适合“无顺序、有放回”的选择，常用小球和隔板法推导。
- [[二项系数]] 出现在组合数和二项式展开中，也是抛硬币计数概率的核心。
- “至少”问题常常用补事件更简单，例如至少 1 张 A = 1 - 一张 A 都没有。

## 核心公式

| 类型 | 公式 | 使用场景 |
|---|---|---|
| 加法原则 | $m+n$ | 分类讨论且不重叠 |
| 乘法原则 | $m\times n$ | 分步骤完成 |
| 阶乘 | $n!$ | 排列组合基础 |
| 排列 | $A(n,k)=\frac{n!}{(n-k)!}$ | 有顺序、无放回 |
| 组合 | $\binom{n}{k}=\frac{n!}{k!(n-k)!}$ | 无顺序、无放回 |
| 有顺序有放回 | $n^k$ | 密码、重复抽样 |
| 无顺序有放回 | $\binom{n+k-1}{k}$ | 可重复组合 |
| 二项式定理 | $(x+y)^n=\sum_{k=0}^n\binom{n}{k}x^ky^{n-k}$ | 二项系数 |
| 公平硬币恰好 $k$ 次正面 | $\frac{\binom{n}{k}}{2^n}$ | 等可能序列 |
| 非公平硬币恰好 $k$ 次正面 | $\binom{n}{k}p^k(1-p)^{n-k}$ | 二项分布雏形 |

## 例题脉络

- 加法原则：选水果、骰子中奇数或大于 4。
- 乘法原则：穿衣搭配、抛硬币、密码。
- 排列：10 人比赛前三名。
- 组合：10 人中选 3 人组成委员会。
- 可重复组合：3 种口味选 5 个冰淇淋球。
- 抛硬币：用组合数计算恰好 $k$ 次正面。
- 扑克牌：5 张牌恰好 2 张 A、至少 1 张 A。
- 生日问题：用补事件计算至少两人生日相同。

## 易错点

- 顺序不重要时误用排列。
- 顺序重要时误用组合。
- 忘记是否放回。
- 直接相加有重叠的分类。
- 遇到“至少”问题时硬算，而没有优先考虑补事件。
