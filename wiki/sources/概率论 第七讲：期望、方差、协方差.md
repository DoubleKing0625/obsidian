# 概率论 第七讲：期望、方差、协方差

原始资料：[[raw/articles/概率论_第七讲_期望方差协方差_完整版|概率论_第七讲_期望方差协方差_完整版]]

## 简短总结

前两讲学会了用 PMF / PDF **描述分布**，这一讲提炼分布的几个**核心数值特征**：[[期望]] 描述中心位置、[[方差]] 描述波动大小、[[协方差]] 描述两个变量共同变化的方向。

## 关键论点

- [[期望]] 是长期平均值：离散用求和 $\sum_x xP(X=x)$，连续用积分 $\int x f_X(x)\,dx$。期望不一定是可取到的值。
- [[期望的线性性]] $E[aX+bY+c]=aE[X]+bE[Y]+c$ **不要求独立**，是二项分布期望 $np$ 的推导关键。
- [[方差]] $Var(X)=E[(X-E[X])^2]$，常用 $E[X^2]-(E[X])^2$ 计算；平方是为了不让正负偏差抵消。
- [[标准差]] $\sigma=\sqrt{Var(X)}$ 与原变量单位一致。
- [[方差]] 性质：$Var(X+c)=Var(X)$、$Var(aX+b)=a^2Var(X)$。
- [[协方差]] $Cov(X,Y)=E[XY]-E[X]E[Y]$，且 $Var(X+Y)=Var(X)+Var(Y)+2Cov(X,Y)$；$Cov(X,X)=Var(X)$。
- [[相关系数]] $\rho=\frac{Cov(X,Y)}{\sigma_X\sigma_Y}\in[-1,1]$，只刻画线性关系。
- [[独立与不相关]]：独立 ⇒ 不相关，但不相关 ⇏ 独立（反例 $X\sim Uniform(-1,1),\,Y=X^2$）。

## 常见分布的期望和方差

| 分布 | 期望 | 方差 |
|---|---|---|
| $Bernoulli(p)$ | $p$ | $p(1-p)$ |
| $Binomial(n,p)$ | $np$ | $np(1-p)$ |
| $Geometric(p)$ | $\frac{1}{p}$ | $\frac{1-p}{p^2}$ |
| $Poisson(\lambda)$ | $\lambda$ | $\lambda$ |
| $Uniform(a,b)$ | $\frac{a+b}{2}$ | $\frac{(b-a)^2}{12}$ |
| $Exponential(\lambda)$ | $\frac{1}{\lambda}$ | $\frac{1}{\lambda^2}$ |
| $Normal(\mu,\sigma^2)$ | $\mu$ | $\sigma^2$ |
