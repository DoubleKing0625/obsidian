# Wiki

这里是由 LLM 维护的知识层。

随着新资料被摄入、问题被探索，Codex 可以在这里创建和更新页面。

## 子目录

- [[wiki/index|索引]]：wiki 的内容地图。
- [[wiki/log|日志]]：wiki 的操作日志。
- [[wiki/sources/README|sources]]：已摄入资料的摘要页。
- [[wiki/concepts/README|concepts]]：可复用的概念、模型、术语和框架。
- [[wiki/entities/README|entities]]：人物、组织、产品、项目、地点等实体。
- [[wiki/topics/README|topics]]：组织相关页面的主题枢纽。
- [[wiki/questions/README|questions]]：值得长期追踪的问题和调查线索。
- [[wiki/syntheses/README|syntheses]]：更高层次的结论、对比、综述和报告。
