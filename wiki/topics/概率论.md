# 概率论

概率论研究不确定性。当前这个主题页覆盖 MITx / MIT 6.041SC 学习路线的前七讲。

## 当前学习地图

1. [[概率模型]]：把现实问题转成 $(\Omega, P)$。
2. [[随机试验]]：描述试验前结果不确定但结果集合已知的情境。
3. [[样本空间]]：列出所有可能基本结果。
4. [[事件]]：把关心的问题表示成样本空间中的子集。
5. [[概率律与概率公理]]：给事件分配概率，并约束概率系统。
6. [[等可能模型]]：在基本结果等可能时用计数求概率。
7. [[条件概率]]：在新信息下重新计算概率。
8. [[乘法公式]]：把条件概率连接到联合概率。
9. [[全概率公式]]：把多个路径导致同一结果的概率加总。
10. [[贝叶斯公式]]：看到结果后反推原因。
11. [[独立性]]：一个事件是否发生不改变另一个事件的概率。
12. [[互斥事件]]：两个事件不能同时发生，和独立性不同。
13. [[两两独立与相互独立]]：多个事件独立性需要额外条件。
14. [[条件独立]]：给定某个条件后，两个事件独立。
15. [[计数方法]]：在等可能模型中计算有利结果数和总结果数。
16. [[排列]] 与 [[组合]]：判断顺序是否重要。
17. [[二项系数]]：组合数、二项式定理和抛硬币计数概率。
18. [[随机变量]]：把随机结果映射成数字的函数。
19. [[离散随机变量]]：取值有限或可数无限的随机变量。
20. [[概率质量函数 PMF]]：离散随机变量的分布描述。
21. [[Bernoulli 分布]]、[[Binomial 分布]]、[[Geometric 分布]]、[[Poisson 分布]]：常见离散分布。
22. [[连续随机变量]]：取值落在区间上、单点概率为 0 的随机变量。
23. [[概率密度函数 PDF]] 与 [[累积分布函数 CDF]]：连续随机变量的分布描述。
24. [[均匀分布]]、[[指数分布]]、[[正态分布]]：常见连续分布。
25. [[无记忆性]]：Geometric 与指数分布共有的“等待不积累”性质。
26. [[期望]]：随机变量的长期平均值（中心位置）。
27. [[期望的线性性]]：$E[aX+bY+c]=aE[X]+bE[Y]+c$，不要求独立。
28. [[方差]] 与 [[标准差]]：描述波动大小。
29. [[协方差]] 与 [[相关系数]]：描述两个变量共同变化的方向与强度。
30. [[独立与不相关]]：独立 ⇒ 不相关，但反之不成立。

## 当前核心直觉

- 概率题第一步通常不是计算，而是建模。
- 样本空间定义错，后面的概率计算都会错。
- 条件概率的本质是“样本空间缩小”。
- 贝叶斯公式的本质是“结果出现后，重新分配原因的可信度”。
- 独立性的本质是“知道一个事件发生，不改变另一个事件的概率”。
- 互斥和独立是不同概念；有正概率的互斥事件通常不独立。
- 计数方法的本质是把等可能概率转化为“有利结果数 / 总结果数”。
- 随机变量的本质是把随机结果转成数字，从而用分布系统描述概率。
- 离散看“点的概率”（PMF、求和），连续看“区间的面积”（PDF、积分）；连续随机变量单点概率为 0。
- 期望的线性性对任意随机变量成立；方差相加才需要协方差为 0。
- 独立比不相关更强：独立一定不相关，不相关不一定独立。
