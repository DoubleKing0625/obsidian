# 原始资料

这里存放原始资料。

把这个目录视为知识库的证据层：

- 尽量保留资料的原始形态。
- 优先新增文件，而不是改写旧文件。
- 下载的图片和附件放到 [[raw/assets/README|assets]]。
- 让 Codex 在 `wiki/` 中总结和整合资料，不要直接把最终知识写在 `raw/` 里。

## 子目录

- [[raw/articles/README|articles]]：网页文章、essay、剪藏页面。
- [[raw/books/README|books]]：书籍、章节、长文阅读资料。
- [[raw/meetings/README|meetings]]：会议、访谈、对话和转录文本。
- [[raw/notes/README|notes]]：零散笔记、粘贴片段、尚未分类的资料。
- [[raw/assets/README|assets]]：图片、PDF、媒体和附件。
